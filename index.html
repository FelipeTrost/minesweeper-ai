Minesweeper solver

Hey, this is my attempt to code a Minesweeper solver. I just look each cell individually and apply very simple logic, if that doesn't work, I identify disjoint groups of tiles (with Union Find a data structure I find pretty cool) and use a backtracking algorithm to find a plausible solution, since there are more than one and there can only be one correct solution this doesn't always get it right. Finally if nothing else works we just pick a random tile.
